Dayflow is a modern, scalable, and secure **Human Resource Management System** designed to digitize and streamline daily HR operations such as employee management, attendance tracking, leave management, and payroll visibility.

This project is built with a **strong focus on database design, backend logic, security, scalability, and clean UI**, keeping hackathon evaluation criteria in mind.

Problem Statement
In many organizations, HR data is scattered across multiple tools or managed manually, leading to inefficiency, data inconsistency, and poor visibility.
Dayflow solves this by providing a single, unified platform** where employees, HR, and admins can manage all HR-related activities seamlessly.

Key Features

 Authentication & Authorization
- Secure login using email and password
- JWT-based authentication
- Role-based access (Admin / HR / Employee)

 Employee Management
- Employee profile management
- Role separation between employees and admins
- Secure access to personal data

 Attendance Management
- Real-time check-in and check-out
- Daily attendance records stored in MySQL
- Secure API-based updates

 Leave Management
- Employees can apply for leave
- Leave status: Pending / Approved / Rejected
- Structured approval workflow

 Payroll (Read-only for Employees)
- Secure payroll visibility
- Data protected at backend level

 Security
- Password hashing using bcrypt
- JWT token-based API protection
- No direct database access from frontend

 Blockchain-Ready Logic
- Audit logs and hash-based integrity design
- Future-ready for blockchain integration
