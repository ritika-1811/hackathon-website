Human Resource Management System (HRMS)
Problem Statement
Managing employees, attendance, leave requests, and payroll manually is time-consuming and error-prone.  
Small and medium organizations need a simple, digital solution to manage core HR operations efficiently.

Solution
This project is a web-based Human Resource Management System (HRMS)** developed during an **8-hour online hackathon
It provides a centralized platform for HR/Admin and Employees to manage daily HR activities such as employee records, attendance, leave management, and payroll.

 Features
 Admin (HR)
- View HR dashboard statistics  
- Add and view employees  
- View and approve/reject leave requests  
- Track attendance records  
- Generate payroll details  

 Employee
- Login to the system  
- Apply for leave  
- View attendance status  
- Access basic payroll information  

 Tech Stack
- *Frontend:* HTML, CSS, JavaScript  
- *Backend:* Python (Flask)  
- *Database:* SQLite  
- *API Type:* REST APIs  
- *Version Control:* GitHub  

 System Architecture
1. User interacts with the frontend UI  
2. Frontend sends requests using JavaScript fetch()  
3. Flask backend processes requests  
4. Data is stored and retrieved from SQLite database  
5. Response is sent back to frontend
